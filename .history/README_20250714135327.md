## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/otp-verification-screen.git
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```
